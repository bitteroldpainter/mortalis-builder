# Road Rage — Chop Shop

Clean modular skeleton for the Road Rage vehicle builder.

## Included in v0.1

- Chassis selection: Bike, Trakk, Buggy and Trukk
- Vehicle naming
- Live summary and point total
- New, Save and Load controls
- Versioned JSON vehicle format
- Responsive interface based on Fighter Forge's visual language
- Separate rules, builder, storage and card modules

## Deliberately not included yet

- Final chassis profiles or costs
- Weapons and weapon mounts
- Hull, engine and crew upgrades
- Portrait cropper
- Vehicle card rendering

All chassis values are currently neutral placeholders so no Road Rage rules have been invented during implementation.

## Run

Open `index.html` in a browser. No server or build step is required.


## v0.2 change

Chassis cards now display Drive, Shoot, AP, Handling, DEF, Hull, Ram and Transport directly in the chooser. Unfinalized values show as an em dash rather than an invented number.
